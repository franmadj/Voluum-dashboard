## Requirements

PHP >= 7.3
BCMath PHP Extension
Ctype PHP Extension
Fileinfo PHP Extension
JSON PHP Extension
Mbstring PHP Extension
OpenSSL PHP Extension
PDO PHP Extension
Tokenizer PHP Extension
XML PHP Extension

## Installation

From root folder:

cp .env.example .env

composer install

php artisan key:generate

php artisan migrate

npm install

npm run dev
